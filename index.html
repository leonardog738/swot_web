01 — Problema

Enchentes matam por falta de sistema

Não é falta de dados — é falta de organização e acesso à informação certa na hora certa.

Sem histórico

Nenhum registro estruturado de alagamentos por setor. Decisões são tomadas no escuro.

Sem previção

Evacuações dependem de feeling ou mensagens informais no WhatsApp.

Dados inacessíveis

APIs gratuitas do INPE e Copernicus existem, mas permanecem sem uso pelo poder público.

Gestão manual

Defesa Civil opera com planilhas e recados em situações críticas de emergência.

02 — Tecnologia

Stack do Space Connect

Tecnologias abertas e gratuitas integradas para monitoramento hidrológico em tempo real.

Python

Linguagem principal para ciência de dados, automação do pipeline e modelos preditivos.

earthaccess

Biblioteca oficial da NASA para autenticação e download de dados via Earthdata.

Hydrocron API

API do PO.DAAC para séries temporais hidrológicas do SWOT em CSV e GeoJSON.

PostGIS

Armazenamento e análise geoespacial avançada dos dados hidrológicos coletados.

06 — Aplicação no dia a dia

Como funciona na prática

Do satélite ao celular do gestor.

1

Satélite passa sobre o Brasil

O SWOT mede o nível dos rios e Lagos com resolução de 250 m a cada ciclo.

2

Pipeline coleta os dados

O sistema baixa e processa automaticamente os dados da API da NASA.

3

Modelo calcula o risco

Algoritmo cruza nível do rio, histórico e previsão para gerar probabilidade de enchente.

5

Ação preventiva é tomada

Equipes são posicionadas e moradores evacuados antes que a água chegue.
